# 06. 开发阶段路线图

## 当前状态

更新时间：2026-05-01

- 阶段 0-3 已完成并推送：产品定版、项目骨架、Sub2API 一键导入、文生图 Alpha。
- CI 已建立：GitHub Actions 覆盖 lint、typecheck、test、Prisma validate、build。
- 阶段 4 已完成主要收口项；阶段 5 已完成任务状态、取消、重新生成、partial preview 和用户并发限制，任务队列待补；阶段 6 遮罩编辑已完成主要闭环；阶段 7 历史、素材库、相册和详情页已完成并通过 CI；阶段 8 商业模板、Prompt 助手和 Prompt 收藏已完成并通过 CI；当前推进阶段 9：运营、计费、风控和社区最小发布。

## 阶段 0：产品定版

目标：开发前锁定边界。

功能/产出：

- 独立站，不嵌入 Sub2API 后台。
- 确认 MVP 使用 Images API。
- 确认一键导入使用短期 ticket。
- 确认密钥不进 URL。
- 确认版本边界、验收标准和可砍功能。

## 阶段 1：项目骨架

目标：站点能跑起来。

功能：

- 前端项目骨架。
- 后端/BFF 项目骨架。
- 环境配置。
- 基础路由：创作台、历史、设置。
- 手动 API Base URL / API Key 配置。
- 基础错误处理。

建议版本：`v0.1-alpha`

## 阶段 2：Sub2API 一键导入

目标：核心集成跑通。

功能：

- Sub2API 创建 launch ticket。
- PsyPic 导入 exchange。
- PsyPic session。
- 后端代持密钥。
- 导入默认模型、可用模型、限制信息。
- 手动导入兜底。

建议版本：`v0.2-alpha`

## 阶段 3：文生图 Alpha

目标：用户能稳定完成文生图，作为内部试用版本。

功能：

- `/v1/images/generations`。
- Prompt 输入。
- 参数面板。
- 尺寸预设和自定义校验。
- 质量、格式、压缩、数量。
- 结果网格。
- 下载、复制、重试。
- 基础历史记录，MVP 可先落 IndexedDB。
- 基础 usage 展示。

建议版本：`v0.3-alpha`

## 阶段 4：图生图与公开试用 MVP

目标：完成基础编辑闭环，并达到公开试用验收口径。

功能：

- 图片上传。
- 粘贴和拖拽。
- 单参考图。
- `/v1/images/edits`。
- 结果转参考图。
- 图生图任务详情。
- 商业尺寸预设，单次只选择一个目标尺寸生成/下载。
- MVP 验收清单回归。

建议版本：`v0.4-beta`

## 阶段 7：历史、素材库、相册

- 中文 prompt 优化。

- Prompt 收藏。
- [x] 模板列表和结构化 Prompt 渲染 BFF API。
- [x] 创作台模板字段编辑。
- [x] 中文 Prompt 优化和图生图主体保留约束。
- [x] Prompt 收藏和一键套用。

## 阶段 9：运营、计费、风控、社区最小发布

目标：可公开上线，并建立社区最小发布和审核对象模型。

功能：

- 成本预估。
- 实际 usage 记录。
- Sub2API 图片计费优化。
- 用户最大 `n`。
- 最大尺寸限制。
- 最大上传大小。
- 用户并发限制。
- 管理端开关。
- 日志审计。
- 最小作品发布模型：私有、链接可见、公开社区。
- 基础作品详情：成图、场景标签、模板来源、参数摘要。
- 基础同款生成入口。
- 社区发布审核开关。
- 举报和下架入口。
- [x] 当前用户图片用量汇总 API。
- [x] 运行时最大 `n`、上传上限、尺寸层级和功能开关服务。
- [x] 生成接口按运行时最大 `n` 拦截。
- [x] 社区公开发布按运行时开关拦截。
- [x] 社区作品 Prisma 长期模型。
- [x] 最小作品发布 API。
- [x] 作品详情 API。
- [x] 同款生成草稿 API 和隐私开关。

建议版本：`v0.9-rc`

## 阶段 10：商业批量、灵感社区、高级工作流、封板

目标：完整产品闭环和社区增长闭环。

功能：

- 批量 prompt。
- CSV 导入。
- 批量多尺寸。
- 批量重试。
- 项目/相册。
- 分享链接。
- 灵感社区信息流。
- 作品详情增强和移动端社区浏览。
- 点赞、收藏、精选、举报的完整体验。
- 一键同款生成增强。
- 从社区作品进入创作台继续编辑。
- 移动端完整适配。
- 压测。
- 安全检查。
- 回滚方案。

建议版本：`v1.0`
