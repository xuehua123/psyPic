# 01. 产品范围

## 产品定位

PsyPic 是一个独立 AI 图片工作台，面向使用 Sub2API 额度/密钥的用户，提供专业、漂亮、稳定的 GPT Image 生成、编辑、版本管理、素材管理和商业复用体验。产品目标不是复刻 Liblib 的大模型生态，也不是只做单张图片生成器，而是先赢项目级商业创作台，再把高质量作品、模板和同款生成沉淀成灵感社区。

一句话定位：

> 面向 Sub2API 用户的商业 GPT Image 创作台和灵感社区，支持文生图、图生图、遮罩编辑、Board Mode（多素材多图层画布）、商业模板、历史素材管理、批量任务、作品发布和同款生成。

工作台化定位：

> PsyPic 的默认首屏应是 Project Workspace：左侧组织项目和创作会话，中间承载图片画廊/对话式版本流，右侧承载参数、模板、参考图、素材和任务详情。

## 工作台对象边界

后续产品、API 和数据模型统一使用以下对象：

- Project：商业项目、客户项目、活动项目或个人创作项目。
- Session / Conversation：围绕一个创作目标持续生成、编辑、分叉的上下文。
- Version Node：一次生成、图生图、遮罩编辑、同款生成或批量展开产生的可回溯节点。
- Asset：结果图、参考图、mask、缩略图、导出 ZIP、社区发布主图，以及可分类复用的素材（background、subject、decoration、text、text_effect、texture_light、shape、brand、reference、mask）。
- Board：project/session 下的 Board Mode 画布，承载多素材编排、手绘标注、mask 草图和导出参考图。
- Job / Task：生成、流式预览、批量任务、导出、素材同步、审核等后台任务。

## 用户类型

### 普通创作者

需求：

- 输入中文描述快速生成图片。
- 想要简单参数，不想理解 API 细节。
- 需要下载、收藏、继续改图。

### 运营/电商用户

需求：

- 批量生成产品主图、海报、封面。
- 保持产品主体一致。
- 多尺寸导出。
- 成本可控。

### 设计/开发用户

需求：

- 精细调参数。
- 查看实际 API 参数和 usage。
- 保存 prompt 版本。
- 使用参考图和遮罩局部编辑。

### Sub2API 管理员

需求：

- 控制是否开放生图站。
- 控制用户额度、倍率、并发、最大尺寸。
- 监控成本、失败率和异常请求。

### 社区浏览/发布用户

需求：

- 浏览商业灵感、作品案例和模板。
- 收藏、点赞、按场景筛选作品。
- 从作品一键同款生成或进入模板创作。
- 发布作品时可选择是否公开 prompt、参数和参考图。

## MVP 必须包含

对应 `v0.4` 公开试用版本。

- 独立站基础框架。
- 手动 API 配置。
- Sub2API 一键安全导入。
- 文生图。
- 图生图/参考图。
- 基础参数面板。
- 商业尺寸预设，单次只选择一个目标尺寸生成/下载。
- 结果预览、下载、重试、继续编辑。
- 基础历史记录。
- 基础 usage/成本展示。
- 错误处理和超时控制。

## V1 必须包含

- 遮罩编辑器。
- 流式 partial preview。
- 任务队列。
- 历史搜索、收藏、标签。
- 商业模板库。
- Prompt 助手。
- 基础作品发布、作品详情和同款生成。
- 后端代理代持密钥。
- 管理端开关、限流、最大尺寸和最大 `n`。
- 更准确的 Sub2API 图片计费。

## V1 后增强

- 批量 prompt 队列。
- CSV 导入。
- 项目/相册升级为完整 Project Workspace。
- 版本流、分支和参数恢复进入服务端可同步模型。
- Board Mode 支持多素材编排、手绘、mask、reference PNG 导出和版本快照。
- 全局搜索覆盖项目、会话、素材、prompt、标签、模板、任务和 request id。
- 灵感社区信息流。
- 公开分享、点赞、收藏、举报、精选。
- 团队空间。
- 品牌资产库。
- Batch API 成本优化。

## 不在 MVP 范围

- 完整社区广场不在 MVP，但必须纳入最终封板路线。
- 公开模型市场。
- 复杂工作流编排。
- 自建模型训练。
- 视频生成。
- 第三方支付和完整会员系统。
- 多租户团队权限。

这些功能可以在核心创作闭环稳定后逐步加入。

## 成功标准

MVP 成功标准：

- 用户能从 Sub2API 一键打开 PsyPic，并直接生成图。
- 文生图和图生图成功率稳定。
- 用户能查看历史并继续编辑上一张图。
- 关键错误信息可理解。
- 密钥没有出现在 URL、日志、浏览器明文长期存储中。

最终封板标准：

- 创作、编辑、管理、成本、风控、移动端、压测全部通过。
- 商业创作台和灵感社区形成闭环：看案例、同款生成、编辑、发布。
- 普通用户不需要理解 API 参数也能高质量出图。
- 高级用户可以完整控制 `gpt-image-2` 参数。
- 管理员能控制成本和风险。
