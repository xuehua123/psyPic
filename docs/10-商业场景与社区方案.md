# 10. 商业场景与社区方案

## 定位

PsyPic 不做 Liblib 的大而全复刻，但最终必须同时具备商业创作台和灵感社区。

核心策略：

- 商业创作台负责高效产图、控成本、控参数、控素材。
- 灵感社区负责发现案例、沉淀模板、带动同款生成和传播。
- 社区内容必须能回流创作台，不能只做浏览信息流。
- 作品、模板和同款草稿都应挂在 project / session / version node 上，而不是只是一条孤立发布记录。
- 拼图画布负责把多张商业素材、参考图、手绘修改意图和 mask 草图组织成一次可生成的视觉 brief。

## 商业场景覆盖

### MVP 必须覆盖

- 电商主图。
- 产品换背景。
- 商品场景图。
- 广告 Banner。
- 小红书/公众号/短视频封面。
- 人像头像和形象照。
- 结果继续编辑。
- 基础尺寸预设和单目标尺寸导出，单次只选择一个尺寸，不做批量多尺寸。

### V1 强化

- 品牌海报。
- 应用/网站首屏视觉。
- 包装和贴图 mockup。
- 节日活动海报。
- 店铺装修素材。
- 详情页局部图。
- 批量同款多尺寸。

### 封板目标

- CSV 批量生成。
- 同一商品多风格、多尺寸、多平台适配。
- 品牌资产库。
- 项目/相册归档。
- 团队分享和审稿。
- 成本、失败率、耗时和 request id 可追踪。

## 商业模板体系

模板必须是结构化表单，不只是 prompt 文本。

每个模板至少包含：

- 场景：电商主图、广告、封面、海报、头像等。
- 主体：产品、人像、品牌物料、IP 形象。
- 画面目标：卖点、氛围、平台、比例。
- 风格：真实摄影、商业海报、极简、潮流、节日等。
- 约束：不改变主体、不改 logo、不乱加文字、不改变人物身份。
- 输出：尺寸、数量、格式、是否透明背景、是否压缩。

模板应用后还应携带工作台上下文：

- 所属 project。
- 当前 session。
- 来源 version node。
- 当前节点的参数快照和参考图快照。
- 可选 board document / board snapshot，用于保留拼图画布中的构图和修改意图。

## 灵感社区

社区的第一目标不是社交，而是提高商业创作效率。

### 信息流

- 推荐、最新、热门、精选。
- 按商业场景筛选。
- 按尺寸、风格、用途、行业筛选。
- 卡片必须有同款入口。

### 作品详情

- 成图展示。
- 场景和标签。
- 作者和发布时间。
- 模板来源。
- 参数摘要。
- 是否公开 prompt、参数、参考图。
- 一键同款生成。
- 作为参考图继续编辑。
- 显示所属 project / session / version node 的最小上下文，让用户知道这张图在工作台里的位置。

### 发布与隐私

- 默认私有。
- 可选链接可见。
- 可选公开社区。
- Prompt 默认隐藏。
- 参考图和原始素材默认隐藏。
- 发布前提示商业素材、版权、肖像权、商标和客户保密风险。
- 发布记录应继承当前 project/session 的隐私默认值，只有用户显式改动才提升公开范围。

### 社区互动

- 点赞。
- 收藏。
- 关注。
- 举报。
- 精选。
- 下架。

评论可以后置，不作为封板前必须项。

## 阶段安排

### `v0.4` 公开试用 MVP

- 创作台可用。
- 商业场景入口可见。
- 基础历史和继续编辑可用。
- 基础尺寸预设和单目标尺寸导出可用。
- 不做完整社区。

### `v0.8` 商业模板

- 商业模板中心。
- Prompt 助手按商业场景生成结构化 prompt。
- 结果可保存为个人模板。

### `v0.9` 运营与审核

- 最小作品发布模型：私有、链接可见、公开社区。
- 基础作品详情：成图、场景标签、模板来源、参数摘要。
- 基础同款生成入口。
- 管理端控制社区发布开关。
- 举报、下架、精选。
- 成本和风控数据可追踪。
- 作品发布、同款草稿和回到创作台的路径都要保留 project/session 语义。

### `v1.0` 封板

- 灵感社区信息流。
- 作品详情增强。
- 点赞、收藏、精选、举报完整体验。
- 一键同款生成增强。
- 社区作品回到创作台继续编辑。
- 移动端浏览和创作主流程完整。

## 不做或后置

- 公开模型市场。
- 自建模型训练。
- 复杂工作流市场。
- 第三方支付和完整会员系统。
- 评论区深度社交。
- 创作者收益分成。

这些功能可以在商业创作台和灵感社区闭环稳定后再评估。

## 验收标准

- 商业用户能在 3 分钟内从模板开始生成第一张可用图。
- 每张公开社区作品都能一键进入同款生成。
- 用户能明确控制 prompt、参数、参考图和素材是否公开。
- 管理员能下架社区作品并关闭公开发布。
- 商业模板至少覆盖电商主图、产品换背景、广告 Banner、社媒封面、品牌海报。
