# 07. 验收清单

## MVP 验收

对应 `v0.4` 公开试用版本。

- [x] 独立站可部署。
- [ ] 手动配置 Sub2API Base URL 和 API Key 后可以生图。
- [ ] 从 Sub2API 一键导入可用。
- [x] 密钥不出现在 URL。
- [x] 文生图可用。
- [ ] 图生图可用。
- [x] 支持 `gpt-image-2` 默认模型。
- [ ] 支持尺寸预设和自定义尺寸校验。
- [ ] 支持商业尺寸预设，单次只选择一个目标尺寸生成/下载。
- [x] 支持 `quality`。
- [x] 支持 `output_format`。
- [ ] 支持 JPEG/WebP `output_compression`。
- [x] 禁止 `background=transparent`。
- [x] 不传 `input_fidelity`。
- [x] 支持结果下载。
- [x] 支持失败重试。
- [x] 支持基础历史。
- [x] 显示 request id、耗时、usage。
- [ ] 超时、429、401、403、400 错误有明确提示。

- [ ] 文生图、图生图、遮罩编辑、流式生成完整可用。
- [ ] Sub2API 一键导入安全可用。
- [ ] 后端代持密钥或等效安全方案可用。

- [ ] 用户级限流、额度、并发限制可用。

- [ ] 压测通过。

## 测试场景

### 正常场景

- [x] 生成 1 张 `1024x1024` PNG。
- [ ] 生成 4 张 `1024x1024` JPEG。
- [ ] 生成 `1536x1024` high。
- [ ] 生成 `3840x2160` high，若管理员允许。
- [ ] 上传 1 张参考图生成。
- [ ] 上传多张参考图生成。
- [ ] 使用 mask 局部编辑。
- [ ] 流式生成并收到 partial image。

### 参数错误

- [ ] 自定义尺寸不是 16 倍数时自动规整。
- [ ] 超过最大边时报错或规整。
- [ ] 长宽比超过 3:1 时拦截。
- [x] `output_compression` 在 PNG 下隐藏或不传。
- [x] `background=transparent` 被拦截。
- [x] `input_fidelity` 不被传递。

### 权限和额度

- [ ] ticket 过期。
- [ ] ticket 重复消费。
- [ ] key inactive。
- [ ] key quota exhausted。
- [ ] key 不属于当前用户。
- [ ] group 不是 OpenAI。

### 稳定性

- [x] 上游 429。
- [ ] 上游 500。
- [ ] 生成超时。
- [ ] 用户取消任务。
- [ ] 上传大文件。
- [ ] 网络中断后恢复。
